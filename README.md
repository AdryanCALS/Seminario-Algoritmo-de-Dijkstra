# Seminário: Algoritmo de Dijkstra

Este repositório contém os materiais e códigos utilizados para a apresentação sobre o **Algoritmo de Dijkstra**, desenvolvido para a disciplina de Estrutura de Dados e Algoritmos. 

## Sobre o Projeto

O objetivo deste projeto é apresentar de forma clara e interativa o funcionamento do algoritmo de caminho mais curto de Dijkstra. O repositório inclui:

*   **Implementação em Python:** Código-fonte demonstrando o algoritmo com e sem o uso de fila de prioridade, destacando as diferenças de performance e complexidade[cite: 1].
*   **Jupyter Notebook:** Um notebook interativo (`main.ipynb`) contendo a explicação teórica, demonstração passo a passo do algoritmo e visualizações gráficas de sua execução em tempo real[cite: 1].
*   **Recursos Visuais:** Imagens, diagramas e animações utilizados para ilustrar conceitos fundamentais, como grafos, arestas e a lógica de processamento dos nós[cite: 1].

## Estrutura do Repositório

*   `/`: Diretório principal contendo o notebook do seminário.
*   `/resources`: Pasta contendo todas as imagens e diagramas utilizados no projeto e na apresentação[cite: 1].
*   `requirements.txt`: Lista de dependências e bibliotecas Python necessárias para executar o notebook corretamente[cite: 1].

## Como Executar

1. Clone o repositório para sua máquina local.
2. Certifique-se de ter o Python instalado.
3. Instale as dependências executando: `pip install -r requirements.txt`[cite: 1].
4. Inicie o Jupyter Notebook ou abra o arquivo `main.ipynb` no VS Code (ou sua IDE de preferência)[cite: 1].
